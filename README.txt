A sample re-distribution of python 2.7.9 for Windows 7.

Content:
Python2.7.rar - The executables for Python 2.7.9
envpython.bat - Environment setup for Python 2.7.9

Installation:
1. Extract the rar and make a note of the fullpath where python.exe is present.
2. Update the noted path to PythonDIR variable in envpython.bat

How to use:
1. In windows start type cmd
2. Goto path where envpython.bat is present. Execute it
3. Type python. You should be able to see python command.
